# GameVocal Godot Plugin

Connects Godot projects to GameVocal and imports dialogue audio, localization, lip-sync data, and integration metadata.

## Setup

1. Copy the `addons/gamevocal` folder to your Godot project's `addons` folder.
2. Enable the plugin in Project Settings -> Plugins.
3. Configure your GameVocal API Key (stored securely in editor settings, not project files).

## Supported Godot Versions

- Godot 4.7+ (Fully compatible with Godot 4.7)

## Character Mapping & Lip-Sync

GameVocal provides a AAA-grade lip-sync system out of the box, perfectly synchronized to your audio playback. It uses ARKit52 blendshapes and includes an auto-mapping feature to wire up your custom 3D models instantly.

### Step-by-Step Guide

Follow these steps to set up fully synchronized facial animations in your Godot scene:

#### 1. Add the Node
- In your Godot scene, click the **+ (Add Node)** button.
- Search for and add a `GameVocalCharacter` node to your scene. (This node handles both audio playback synchronization and blendshape animation).

#### 2. Set up the Audio Player
- Add an `AudioStreamPlayer`, `AudioStreamPlayer2D`, or `AudioStreamPlayer3D` to your scene (depending on if you want the voice to be spatialized).
- Click on your `GameVocalCharacter` node.
- In the Inspector, click on the **Audio Player** slot and select the AudioStreamPlayer you just created.

#### 3. Assign your 3D Model
- Ensure you have a 3D character in your scene that uses a `MeshInstance3D` with blendshapes (often called shape keys or morph targets in 3D software).
- In the `GameVocalCharacter` Inspector, click the **Target Mesh** slot and assign your character's `MeshInstance3D` node.

#### 4. Auto-Map the Blendshapes
- In the `GameVocalCharacter` Inspector, look for the **Lip-Sync Mapping** section.
- Check the box labeled **Auto Map Blendshapes**. 
- The plugin will instantly scan your 3D model's custom blendshape names and use fuzzy matching (e.g., `Mouth_Smile_Left` -> `mouthSmileLeft`) to wire them up to the ARKit 52 standard. Check your Godot Output console to see how many shapes were successfully matched!

#### 5. Play Synced Dialogue
- To trigger the character speaking, simply call the `play_dialogue` function from any script, providing the paths to the audio file and its matching JSON lip-sync file generated by GameVocal:

```gdscript
# Example from a main game script:
func _ready():
    $GameVocalCharacter.play_dialogue("res://MyProject/Assets/Voice1.ogg", "res://MyProject/Assets/Voice1.json")
```

The node will automatically load the data, play the audio, and smoothly interpolate the facial animations in perfect sync with the playback position!
